Detailed Description

Multi-dimensional index with stride-based traversal.

Efficiently handles indexed access to tensors with various memory layouts. Uses compile-time optimization with constexpr where possible

Moves the tensor's internal position to the specified coordinates.

This function updates the tensor's internal position and offset based on the provided coordinates. It ensures that the coordinates match the tensor's dimensions and are within the valid range of the tensor's shape.

Parameters
coordsA span of integers representing the coordinates to move to. The size of the span must match the number of dimensions of the tensor.
Exceptions

BREZEL_FORCE_INLINE bool brezel::tensor::StridedIndex::next ( )
inlinenoexcept

Advances to the next position in a multidimensional tensor layout.

This function increments the current position within the tensor layout. If the position exceeds the total size of the tensor, it returns false. Otherwise, it updates the coordinates and offset based on the tensor's shape and strides.

Returns
true if the position was successfully incremented and is within bounds.
false if the position exceeds the total size of the tensor.
