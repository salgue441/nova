Constructs a binary expression between two tensor expressions.

Creates a binary operation between two tensor expressions, computing the resulting shape through broadcasting rules.

Template Parameters
TThe data type of the tensor elements
Parameters
leftLeft-hand side tensor expression operand
rightRight-hand side tensor expression operand
opBinary operation to be performed between the tensors
Note
The resulting shape is calculated using broadcasting rules where smaller tensors are broadcast to match the shape of larger ones where possible

Evaluates the binary tensor expression at the given indices.

This function applies the binary operation stored in m_op to the evaluated results of the left and right operands at the given indices. The indices are mapped according to broadcasting rules before evaluation.

Parameters
indicesSpan of indices where the expression should be evaluated
Returns
T Result of applying the binary operation to both operands at the given indices
