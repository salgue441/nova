Constructs a random number generator for numeric types.

Creates a random number generator that produces values in the range [min_val, max_val]. The generator works with both floating point and integral types.

Template Parameters
TThe numeric type (must be either floating point or integral)
Parameters
min_valThe minimum value in the range (inclusive)
max_valThe maximum value in the range (inclusive)
Exceptions
static_assertIf T is not a floating point or integral type
Note
Uses std::uniform_real_distribution for floating point types
Uses std::uniform_int_distribution for integral types

Creates a uniform distribution of either real or integer values based on the template parameter type T. For floating point types, uses uniform_real_distribution. For integral types, uses uniform_int_distribution.
