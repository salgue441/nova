Detailed Description

Represents the shape of a tensor.

Provides a lightweight wrapper around dimensions with validation and common shape operations. The Shape class efficiently represents tensor dimensions and supports broadcasting, reshaping, and other common operators. Small-vector optimization is used to avoid heap allocations for common tensor dimensions (up to 8 dimensions stored inline).
