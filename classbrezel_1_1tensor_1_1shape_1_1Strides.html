Detailed Description

Represents the strides of a tensor.

Handles memory layout and indexing calculation for tensor access. Provides efficient conversion between multi-dimensional and linear indices. Uses small-vector optimization to avoid heap allocations for common tensor ranks (up to 8 dimensions stored inline).
