Detailed Description

Memory pool for efficient allocation and reuse.

Implements a size-based memory pooling strategy to reduce allocation overhead and memory fragmentation. Unused memory blocks are cached for future reuse, improving performance for tensor operations that frequently allocate temporary buffers.
