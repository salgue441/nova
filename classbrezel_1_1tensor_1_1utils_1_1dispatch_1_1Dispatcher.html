Detailed Description

Singleton class responsible for registering standard and binary operations for various types.

The Dispatcher class provides methods to register standard types and binary operations for a variety of types including floating point types, integer types, boolean, and complex types. It ensures that the registration is done in a singleton instance to maintain a single point of control.

Usage:

Dispatcher::instance().register_standard_types<Op>(registry);
Dispatcher::instance().register_binary_standard_types<Op>(binary_registry);

The class is non-copyable and non-movable to ensure the singleton property.

Registers standard types with the given TypeDispatchRegistry.

This function registers a set of standard types (such as float, double, various integer types, boolean, and complex numbers) with the provided TypeDispatchRegistry. It uses the provided template operation (Op) to register functions for each type.
